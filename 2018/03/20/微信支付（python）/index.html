微信支付（python)

说明：本篇博文是作者从github的微信支付开源项目中挑选的一个比较易于理解其原理的项目作说明，并不一定是最有的微信支付工具。github地址

安装：

生成支付链接

  • 请求url：https://api.mch.weixin.qq.com/pay/unifiedorder

  • 生成统一下单请求所需要提交的数据

    • appid

    • mch_id

    • nonce_str：32位随机字符串（uuid：36位，不过有4个-，要全替换为空，uuid.uuid4产生的是位随机数，有可能重复，或者用char = string.ascii_letters + string.digits从中随机出32位随机数，不过有可能重复）

    • sign：将之前的参数生成md5签名参数

    • 其他基本参数（必传,放到**kwargs里面）

      • body：商品详情

      • out_trade_no：商户订单号

      • total_fee：标价金额

      • spbill_create_ip：终端ip，支持IPV4和IPV6两种格式的IP地址。调用微信支付API的机器IP

      • notify_url：通知地址

      • trade_type：交易类型—JSAPI -JSAPI支付NATIVE -Native支付APP -APP支付

+ 其他可选参数见[官网开发文档](<https://pay.weixin.qq.com/wiki/doc/api/native.php?chapter=9_1>)
